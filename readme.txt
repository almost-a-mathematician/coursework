Системные требования
Данное программное средство предназначено для автоматизации поставок

Чтобы у вас работала программа, нужно:
1. иметь компьютер/ ноутбук/ под упрвлением операционной системы windows 7/8.1/8/10 
2. открыть программу main.exe из папки EXE
- процессор: Pentium IV и выше;
- объем оперативной памяти: 512 Мбайт и выше;
- свободного места на жестком диске: 1 Гб;
- операционная система: Windows 7 и выше;
- наличие .NET Framework 4.4 версии и выше;
- наличие манипулятора «мышь»;
- клавиатура IBM PC любой модификации.
